How to use this dashboard

  1. Confirm the Worker has ALLOWED_ORIGINS and ADMIN_TOKEN. The token must use ASCII letters, numbers, and standard keyboard symbols only—no emoji, smart punctuation, or accented characters.
  2. Enter that same ADMIN_TOKEN below and select Load report.
  3. Choose the viewer, add an optional private label, and select Create link.
  4. The personalized link is copied automatically. Paste it into the private message or email you send.
  5. Return here and select Load report to see starts, choices, completions, and second-appointment continuations.

The credential stays in this browser session and is never stored by this page. Reporting excludes email addresses, IP addresses, and browser fingerprints.
